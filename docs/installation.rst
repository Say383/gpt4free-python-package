============
Installation
============


Stable release
--------------

To install gpt4free, run this command in your terminal:

.. code-block:: console

    $ pip install gpt4free

This is the preferred method to install gpt4free, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for gpt4free can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/rzashakeri/gpt4free

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/rzashakeri/gpt4free/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/rzashakeri/gpt4free
.. _tarball: https://github.com/rzashakeri/gpt4free/tarball/master
